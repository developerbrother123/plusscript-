1. Introducing PlusScript

PlusScript is the Hyper-Language—a modern, intuitive programming language built on Python, designed for rapid development across diverse domains.

1.1 Purpose

PlusScript simplifies coding while offering tools for web, mobile, desktop, AI, IoT, blockchain, and quantum computing.

1.2 Features

  • `+`-based syntax for readability.
  • Extensive built-in modules.
  • Cross-platform compatibility.

1.3 Developer Benefits

Expect a language that’s easy to learn, powerful to use, and versatile for any project.

2. Setting Up Your Environment

Get started with PlusScript:

  1. Download `PlusScriptIDE.exe` or source files.
  2. Install Python 3.9+ and libraries: 
    pip install tkinter Pillow numpy psycopg2 mysql-connector-python boto3 paho-mqtt web3
  3. Run `PlusScriptIDE.exe` or `python plusscript_ide.py`.
  4. Test with: 
    show "Hello, PlusScript!"

3. Variables and Data Types

Variables store data using `set`:

set num = 42
set text = "Hello"
set items = [1, 2, 3]
set info = {"name": "Alice", "age": 25}

Types: Integers, floats, strings, lists, dictionaries.

4. Operators and Expressions

Use operators for calculations and comparisons:

set sum = 5 + 3    # Addition
set diff = 10 - 2  # Subtraction
set eq = 5 == 5   # Equality
set gt = 7 > 4    # Greater than

Supported: `+`, `-`, `*`, `/`, `==`, `>`, `<`, `>=`, `<=`.

5. Control Flow

+if x > 0
    show "Positive"
+elif x < 0
    show "Negative"
+else
    show "Zero"
+end

+while x < 5
    show x
    set x = x + 1
+end

+for i in range(3)
    show i
+end

6. Functions and Scope

+func add a b
    return a + b
+end
show add(2, 3)  # 5

+func outer
    set x = 10
    +func inner
        show x
    +end
    inner()
+end
outer()  # 10

Functions use local scope unless referencing outer variables.

7. Classes and Objects

+class Car
    set speed = 0
    +method accelerate
        set self.speed = self.speed + 10
    +end
+end
set mycar = Car()
mycar.accelerate()
show mycar.speed  # 10

8. Modules and Packages

Import with `+import` or install with `pkg`:

+import "math.ps"
pkg "utils" "utils.ps"

9. Error Handling

+try
    set x = 1 / 0
+catch e
    show "Error: " + e
+finally
    show "Done"
+end

10. File I/O

write "data.txt" "Hello"
set content = read("data.txt")
show content  # "Hello"

11. Web Development

Build a simple server:

server 8080
api "hello" "/hello"
set api_endpoints = {"hello": "Hi, Web!"}

12. Android Development

Create an Android app:

android "MyApp"
gui "main" 300 500
set msg = "Android Rocks!"
show msg

13. Desktop Applications

Build a desktop GUI:

desktop "MyApp" 400 600
gui "window" 380 580

14. Game Development

Make a simple game:

game "Pong" 800 600
draw "ball" 400 300

15. Artificial Intelligence

Work with tensors:

ai "model" 10
set data = tensor(1, 2, 3)
show data

16. Internet of Things (IoT)

Control an IoT device:

iot "sensor" "mqtt://broker"
iot_pub("sensor", "temp", "25")

17. Blockchain Integration

Interact with a blockchain:

blockchain "eth" "http://node"
set tx = {"to": "0x...", "value": 1}
block_tx("eth", tx)

18. Scientific Computing

Solve equations:

sci "calc" "2 * x = 10"
set x = sci_solve("calc")
show x  # 5

19. Quantum Computing

Simulate a quantum circuit:

quantum "qubit" "simulate"
set state = [0.707, 0.707]  # Hadamard gate effect
show state

Future integration with Qiskit planned.
